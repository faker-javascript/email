## Install

```
$ npm install --save @fakerjs/email
```

## Usage

```js
import email from '@fakerjs/email';

email();
//=> itiwuhilacupi@panetoc.services

email({domain: 'example.com'})
//=> rad@example.com
```

## Tests

Run tests

```
npm run test
```
